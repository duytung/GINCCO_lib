Welcome to GINCCO_lib's documentation!
======================================

GINCCO_lib is a Python library developed to facilitate the efficient import, analysis, and visualization of results from the SYMPHONIE ocean model. It aims to streamline post-processing workflows for researchers working with numerical ocean modeling data.
